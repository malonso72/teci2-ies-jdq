Onda senoidal y desfase [Ponencia D1]★★★

En una subestación eléctrica de Sevilla se registran las siguientes señales de tensión e intensidad en corriente alterna:

\(v(t) = 325 \cdot \operatorname{sen}(314t)\) V \(\quad i(t) = 10\sqrt{2} \cdot \operatorname{sen}(314t - 45°)\) A

a) Determinar el valor máximo y el valor eficaz de la tensión y de la intensidad.

b) Calcular la frecuencia, el período y la pulsación angular.

c) Determinar el desfase entre tensión e intensidad e indicar si el circuito es inductivo o capacitivo.

d) Escribir la expresión temporal de la intensidad si su valor eficaz fuera 5 A, la misma frecuencia y un desfase de +30° respecto a la tensión.

📝 Ver resolución completa

Apartado a — Valores máximos y eficaces

\[V_{m}\ = \ 325\ V\ \ \Rightarrow \ \ V_{ef}\ = \ V_{m}\ /\ \sqrt{2}\ = \ 325\ /\ 1,414\ = \ 229,8\ V\ \approx \ 230\ V\]

\[I_{m}\ = \ 10\sqrt{2}\ A\ \ \Rightarrow \ \ I_{ef}\ = \ 10\sqrt{2}\ /\ \sqrt{2}\ = \ 10\ A\]

Apartado b — Parámetros de la onda

\[\omega\ = \ 314\ rad/s\ \ \Rightarrow \ \ f\ = \ \omega/(2\pi)\ = \ 314/6,283\ = \ 50\ Hz\ \ \Rightarrow \ \ T\ = \ 1/f\ = \ 0,02\ s\ = \ 20\ ms\]

→ \(V_m = 325\) V · \(V_{ef} = 230\) V · \(I_m = 10\sqrt{2}\) A · \(I_{ef} = 10\) A · \(\omega = 314\) rad/s · \(f = 50\) Hz · \(T = 20\) ms

Apartado c — Desfase y carácter del circuito

\[\varphi\ = \ \varphi_{V}\ - \ \varphi_{I}\ = \ 0{^\circ}\ - \ ( - 45{^\circ})\ = \ + 45{^\circ}\]

\(\varphi = +45° > 0\) ⇒ la tensión adelanta a la corriente ⇒ circuito INDUCTIVO (predominio de \(X_L\)).

Apartado d — Expresión temporal de nueva intensidad

\[I_{ef}\ = \ 5\ A\ \ \Rightarrow \ \ I_{m}\ = \ 5\sqrt{2}\ A\ \approx \ 7,07\ A\]

\[\varphi_{I}\ = \ \varphi_{V}\ - \ \varphi\ = \ 0{^\circ}\ - \ ( + 30{^\circ})\ = \ - 30{^\circ}\ \ \Rightarrow \ \ i(t)\ = \ 7,07\ \cdot \ sen(314t\ - \ 30{^\circ})\ A\]

→ \(\varphi = +45°\) ⇒ inductivo · \(i(t) = 7{,}07 \cdot \operatorname{sen}(314t - 30°)\) A

⚠ Claves: \(V_{ef} = V_m/\sqrt{2}\); \(\omega = 2\pi f = 314\) rad/s ⇒ \(f = 50\) Hz. El desfase \(\varphi = \varphi_V - \varphi_I\): positivo ⇒ inductivo; negativo ⇒ capacitivo. En la expresión \(i(t) = I_m \cdot \operatorname{sen}(\omega t + \varphi_I)\), si la tensión adelanta +30°, la corriente lleva −30°.

Circuito RC serie [Ponencia D1]★★★

Una instalación solar de Huelva dispone de un circuito RC serie con R = 40 Ω y C = 106 µF conectado a una fuente de 230 V eficaces a 50 Hz.

a) Calcular la reactancia capacitiva y la impedancia del circuito.

b) Calcular la intensidad eficaz y las tensiones en R y C.

c) Calcular el factor de potencia y el desfase. Indicar el carácter del circuito.

d) Calcular la potencia activa, reactiva y aparente.

📝 Ver resolución completa

Apartado a — Reactancia capacitiva e impedancia

\[X_{C}\ = \ 1/(\omega \cdot C)\ = \ 1/(314\ \times \ 106 \times 10^{-6})\ = \ 1/0,03328\ = \ 30,05\ \mathrm{\Omega}\ \approx \ 30\ \mathrm{\Omega}\]

\[Z\ = \ \sqrt{R^2\ + \ X_{C^2}}\ = \ \sqrt{40^2\ + \ 30^2}\ = \ \sqrt{1600\ + \ 900}\ = \ \sqrt{2500}\ = \ 50\ \mathrm{\Omega}\]

Apartado b — Intensidad y tensiones parciales

\[I\ = \ V/Z\ = \ 230/50\ = \ 4,6\ A\]

\[V_{R}\ = \ I \cdot R\ = \ 4,6 \times 40\ = \ 184\ V\ \ \ \ \ V_{C}\ = \ I \cdot X_{C}\ = \ 4,6 \times 30\ = \ 138\ V\]

Verificación: \(V = \sqrt{V_R^2 + V_C^2} = \sqrt{184^2 + 138^2} = \sqrt{33856 + 19044} = \sqrt{52900} = 230\) V ✓

Apartados c y d — Factor de potencia y potencias

\[cos\varphi\ = \ R/Z\ = \ 40/50\ = \ 0,8\ \ \Rightarrow \ \ \varphi\ = \ - 36,87{^\circ}\ (capacitivo:\ corriente\ adelanta\ a\ la\ tensión)\]

\[P\ = \ V \cdot I \cdot cos\varphi\ = \ 230 \times 4,6 \times 0,8\ = \ 846,4\ W\ \ \ \ Q\ = \ V \cdot I \cdot sen\varphi\ = \ 230 \times 4,6 \times 0,6\ = \ 634,8\ VAR\]

\[S\ = \ V \cdot I\ = \ 230 \times 4,6\ = \ 1058\ VA\]

→ \(X_C = 30\) Ω · \(Z = 50\) Ω · \(I = 4{,}6\) A · \(V_R = 184\) V · \(V_C = 138\) V · \(\cos\varphi = 0{,}8\) · \(P = 846\) W · \(Q = 635\) VAR · \(S = 1058\) VA

⚠ En RC serie el ángulo \(\varphi\) es negativo (capacitivo): la corriente adelanta a la tensión. El triángulo de impedancias tiene \(R\) en la horizontal, \(X_C\) hacia abajo y \(Z\) como hipotenusa. Comparar con RL donde \(X_L\) va hacia arriba.

Resonancia en paralelo [Ponencia D1]★★★

Un circuito en paralelo formado por una bobina ideal (L = 50 mH) y un condensador (C = 20 µF) está conectado a una fuente de 200 V eficaces de frecuencia variable. En paralelo con ambos hay una resistencia R = 100 Ω.

a) Calcular la frecuencia de resonancia del circuito paralelo.

b) Demostrar que en resonancia la corriente total es mínima e igual a la corriente por R. Calcular su valor.

c) Calcular las corrientes por L y por C en resonancia y verificar que se anulan.

d) Comparar con la resonancia en serie: indicar en qué se parecen y en qué difieren.

📝 Ver resolución completa

Apartado a — Frecuencia de resonancia paralelo

\[Condición:\ X_{L}\ = \ X_{C}\ \ \Rightarrow \ \ \omega _0L\ = \ 1/(\omega _0C)\ \ \Rightarrow \ \ \omega _0\ = \ 1/\sqrt{LC}\]

\[\omega _0\ = \ 1/\sqrt{50 \times 10^{-3}\ \times \ 20 \times 10^{-6}}\ = \ 1/\sqrt{10^{-6}}\ = \ 1000\ rad/s\]

\[f_0\ = \ \omega _0/(2\pi)\ = \ 1000/6,283\ = \ 159,2\ Hz\]

Apartado b — Corriente total mínima en resonancia

\[X_{L}\ = \ \omega _0 \cdot L\ = \ 1000 \times 0,05\ = \ 50\ \mathrm{\Omega}\ \ \ \ \ X_{C}\ = \ 1/(\omega _0 \cdot C)\ = \ 1/(1000 \times 20 \times 10^{-6})\ = \ 50\ \mathrm{\Omega}\]

\[I_{L}\ = \ V/X_{L}\ = \ 200/50\ = \ 4\ A\ (retrasa\ 90{^\circ})\ \ \ \ \ I_{C}\ = \ V/X_{C}\ = \ 200/50\ = \ 4\ A\ (adelanta\ 90{^\circ})\]

\[I_{LC}\ = \ I_{L}\ + \ I_{C}\ = \ 4\angle - 90{^\circ}\ + \ 4\angle + 90{^\circ}\ = \ 0\ \ \Rightarrow \ \ I_{total}\ = \ I_{R}\ + \ 0\ = \ V/R\ = \ 200/100\ = \ 2\ A\ (mínima)\]

Apartado d — Comparación resonancia serie vs. paralelo

Ambas: misma condición \(f_0 = \dfrac{1}{2\pi\sqrt{LC}}\), \(\cos\varphi = 1\), carácter resistivo. Diferencia: serie ⇒ Z mínima, I máxima, sobretensiones en L y C; paralelo ⇒ Z máxima, I mínima, sobrecorrientes en L y C que se anulan.

→ f₀ = 159,2 Hz · Itotal = 2 A (mínima) · IL = IC = 4 A (se anulan) · Paralelo: Z máx, I mín

⚠ Resonancia paralelo: misma \(f_0\) que en serie (\(1/2\pi\sqrt{LC}\)), pero efectos opuestos. En paralelo las corrientes IL e IC son iguales en módulo y opuestas en fase ⇒ se anulan ⇒ Itotal = IR. La impedancia es máxima (circuito casi abierto para la fuente).

Almazara de Baena — Serie RLC: impedancia y tensiones★★★

La instalación eléctrica de una almazara de Baena (Córdoba) alimenta una carga que puede modelarse como un circuito RLC serie formado por una resistencia de 30 Ω, una bobina de 127,3 mH y un condensador de 53,05 μF. El conjunto se conecta a la red monofásica de 230 V eficaces con una frecuencia de 50 Hz.

a) Calcule las reactancias inductiva XL y capacitiva XC, y la impedancia total Z del circuito.

b) Determine la intensidad eficaz que recorre el circuito y las caídas de tensión UR, UL y UC en cada uno de los tres elementos.

c) Obtenga el factor de potencia cos φ e identifique, razonadamente, el carácter inductivo o capacitivo del circuito.

📝 Ver resolución completa

Apartado a — Reactancias e impedancia

\[XL\ = \ 2\pi \cdot f \cdot L\ = \ 2\pi \times 50 \times 0,1273\ = \ 40\ \Omega\]

\[XC\ = \ \frac{1}{2\pi \cdot f \cdot C} = \ \frac{1}{2\pi \times 50 \times 53,05 \times 10^{-6}} = \ 60\ \Omega\]

\[X\ = \ XL - XC\ = \ 40 - 60\ = \ - 20\ \Omega\ \ \rightarrow \ \ capacitivo\]

\[Z\ = \ \sqrt{R^2 + X^2} = \ \sqrt{30^2 + 20^2} = \ \sqrt{1300} \approx \ 36,06\ \Omega\]

→ XL=40 Ω · XC=60 Ω · Z=36,06 Ω

Apartado b — Intensidad y tensiones

\[I\ = \ \frac{U}{Z} = \ \frac{230}{36,06} = \ 6,38\ A\]

\[UR\ = \ 6,38 \times 30\ = \ 191\ V\ \ \ \ UL\ = \ 6,38 \times 40\ = \ 255\ V\ \ \ \ UC\ = \ 6,38 \times 60\ = \ 383\ V\]

Apartado c

\[cos\ \varphi\ = \ \frac{R}{Z} = \ \frac{30}{36,06} = \ 0,832\ \ \rightarrow \ \ CAPACITIVO\ (XC\ > \ XL)\]

⚠ U ≠ UR+UL+UC. Las tensiones están desfasadas y se suman vectorialmente.

⚠ UL y UC pueden ser mayores que U. No es un error.

Polígono PISA (Sevilla) — RLC serie: potencias★★★

En una nave industrial del Polígono PISA (Sevilla) se alimenta a 400 V eficaces y 50 Hz una carga que responde al modelo de un circuito RLC en serie. La resistencia del circuito vale 12 Ω, la reactancia inductiva 20 Ω y la reactancia capacitiva 4 Ω.

a) Calcule la impedancia total Z y el ángulo de desfase φ entre la tensión y la intensidad.

b) Determine las potencias activa P, reactiva Q y aparente S que el circuito demanda de la red.

c) Represente cualitativamente el triángulo de potencias del circuito e indique el tipo de potencia reactiva (inductiva o capacitiva) que consume.

d) Corrija el factor de potencia hasta un valor de 0,95 y calcule la intensidad de línea antes y después de la corrección.

📝 Ver resolución completa

\[X\ = \ 20 - 4\ = \ 16\ \Omega\ \ \rightarrow \ \ inductivo\ \ \ \ Z\ = \ \sqrt{12^2 + 16^2} = \ 20\ \Omega\ \ \ \ I\ = \ \frac{400}{20} = \ 20\ A\]

\[P\ = \ I^2R\ = \ 400 \times 12\ = \ 4800\ W\ \ \ \ Q\ = \ I^2X\ = \ 400 \times 16\ = \ 6400\ VAr\ \ \ \ S\ = \ I^2Z\ = \ 8000\ VA\]

→ \(P = 4{,}8\) kW · \(Q = 6{,}4\) kVAr · \(S = 8\) kVA · \(\cos\varphi = 0{,}6\) · \(S^2 = P^2 + Q^2\): 64=23,04+40,96 ✓

Apartado a — Impedancia y ángulo de desfase

Ya calculados arriba: X = 16 Ω (inductivo) · Z = 20 Ω · φ = 53° · I = 20 A

Ya calculados arriba: X = 16 Ω (inductivo) · Z = 20 Ω · cos φ = 0,6 → φ = 53°

Apartado b — Potencias P, Q y S

Ya calculados arriba: P = 4,8 kW · Q = 6,4 kVAr · S = 8 kVA · cos φ = 0,6

Ya calculados arriba: P = 4,8 kW · Q = 6,4 kVAr · S = 8 kVA

Apartado c — Triángulo de potencias

Ver diagrama a continuación (P horizontal, Q vertical, S hipotenusa; φ = 53°, inductivo).

Apartado d — Corrección del factor de potencia a cosφ'=0,95

Potencia reactiva necesaria con cosφ'=0,95:

\[Q'\ = \ P \cdot \frac{sen\ \varphi'}{cos\ \varphi'} = \ 4800 \times \frac{0,3123}{0,95} = \ 1578\ VAr\]

Potencia reactiva del condensador:

\[Qc\ = \ Q - Q'\ = \ 6400 - 1578\ = \ 4822\ VAr\]

Capacidad del condensador (en paralelo, V=400V, f=50Hz):

\[C\ = \ \frac{Qc}{\omega \cdot V^2} = \ \frac{4822}{2\pi \times 50 \times 400^2} = \ \frac{4822}{50\ 265\ 600} \approx \ 95,9\ µF\]

Nueva corriente de línea:

\[S'\ = \ \sqrt{P^2 + Q'^2} = \ \sqrt{4800^2 + 1578^2} = \ 5051\ VA\ \ \ \ I'\ = \ \frac{S'}{V} = \ \frac{5051}{400} = \ 12,6\ A\]

→ C ≈ 96 µF · I baja de 20,0 A → 12,6 A (−37%) El condensador compensa la reactiva inductiva

⚠ NUNCA sumar P+Q para obtener S. Son vectores: S = \(\sqrt{P^2+Q^2}\).

⚠ Q > 0 inductiva. Q < 0 capacitiva. R no genera Q.

⚠ Corrección fp: condensador en PARALELO (misma tensión). \(C = Q_C/(\omega \cdot V^2)\).

⚠ La potencia activa P no cambia al añadir el condensador.

Boucherot★★

En una pequeña instalación industrial se conectan en paralelo a una red monofásica de 230 V eficaces y 50 Hz tres receptores: un primer receptor (R₁) de 2 000 W con factor de potencia 0,8 inductivo, un segundo receptor (R₂) de 1 500 W con factor de potencia unidad y un tercer receptor (R₃) de 1 000 W con factor de potencia 0,6 capacitivo. Aplicando el teorema de Boucherot, se pide:

a) Calcule la potencia activa, reactiva y aparente totales de la instalación.

b) Determine la corriente total absorbida de la red y el factor de potencia global.

c) Razone si es necesario corregir el factor de potencia e indique qué elemento se añadiría en paralelo para hacerlo.

📝 Ver resolución completa

Apartado a — Boucherot

\[Q_1\ = \ 2000 \times \frac{0,6}{0,8} = \ + 1500\ VAr\ (ind.)\ \ \ \ Q_3\ = \ - 1000 \times \frac{0,8}{0,6} = \ - 1333\ VAr\ (cap.)\]

\[P\ = \ 4500\ W\ \ \ \ Q\ = \ 1500 + 0 - 1333\ = \ + 167\ VAr\ \ \ \ S\ = \ \sqrt{4500^2 + 167^2} \approx \ 4503\ VA\]

→ P=4500 W · Q=+167 VAr · S≈4503 VA

Apartado b

\[I\ = \ \frac{S}{U} = \ \frac{4503}{230} \approx \ 19,6\ A\ \ \ \ cos\ \varphi\ = \ \frac{P}{S} \approx \ 1,0\]

→ I ≈ 19,6 A · cos φ ≈ 1,0 Prácticamente compensado, no necesita corrección

⚠ Boucherot: P se suma directamente. Q se suma CON SIGNO: +inductiva, −capacitiva.

⚠️
NUNCA sumar las S directamente.

Circuito RL serie — Impedancia, potencias y carácter★★★

Un motor eléctrico monofásico de pequeña potencia instalado en un taller de Santa Fe se modela como un circuito RL serie con R = 50 Ω y L = 0,20 H. Se alimenta a 230 V eficaces y 50 Hz.

a) Calcula la reactancia inductiva XL y la impedancia Z del circuito.

b) Calcula la intensidad eficaz I y las tensiones parciales VR y VL. Verifica que \(V = \sqrt{V_R^2 + V_L^2}\).

c) Calcula las potencias activa P, reactiva Q y aparente S. Indica el factor de potencia y el carácter del circuito.

📝 Ver resolución completa

Apartado a — Reactancia inductiva e impedancia

\[XL\ = \ 2\pi \cdot f \cdot L\ = \ 2\pi \times 50 \times 0,20\ = \ 62,83\ \Omega\]

\[Z\ = \ \sqrt{R^2 + XL^2} = \ \sqrt{50^2 + 62,83^2} = \ \sqrt{6453,9} = \ 80,34\ \Omega\ \ \ \ \varphi\ = \ arctan(\frac{XL}{R})\ = \ arctan(\frac{62,83}{50})\ = \ 51,5{^\circ}\]

→ XL = 62,83 Ω · Z = 80,34 Ω · φ = 51,5° · Carácter INDUCTIVO

Apartado b — Intensidad y tensiones parciales

\[I\ = \ \frac{V}{Z} = \ \frac{230\ V}{80,34\ \Omega} = \ 2,864\ A\]

\[VR\ = \ I \cdot R\ = \ 2,864 \times 50\ = \ 143,2\ V\ \ \ \ VL\ = \ I \cdot XL\ = \ 2,864 \times 62,83\ = \ 180,0\ V\]

Verificación vectorial:

\[\sqrt{VR^2 + VL^2} = \ \sqrt{143,2^2 + 180,0^2} = \ \sqrt{20507 + 32400} = \ 230,0\ V\ ✓\]

→ I = 2,864 A ; VR = 143,2 V ; VL = 180,0 V ; VL > VR porque XL > R

Apartado c — Potencias y factor de potencia

\[P\ = \ I^2 \cdot R\ = \ 2,864^2 \times 50\ = \ 410\ W\]

\[Q\ = \ I^2 \cdot XL\ = \ 2,864^2 \times 62,83\ = \ 515\ VAr\ \ (inductiva,\ Q\ > \ 0)\]

\[S\ = \ I^2 \cdot Z\ = \ 2,864^2 \times 80,34\ = \ 659\ VA\ \ \ \ cos\ \varphi\ = \ \frac{P}{S} = \ \frac{410}{659} = \ 0,622\]

→ P = 410 W ; Q = 515 VAr ; S = 659 VA ; cosφ = 0,622 ; Carácter inductivo: φ > 0, la corriente retrasa a la tensión

⚠ RL serie: NO hay condensador → Z = \(\sqrt{R^2 + X_L^2}\). No restar reactancias.

⚠ V ≠ VR + VL (suma aritmética). V \(= \sqrt{V_R^2 + V_L^2}\) siempre en CA.

⚠ Circuito inductivo: corriente retrasada, Q > 0, cosφ < 1. La bobina consume potencia reactiva.

Paralelo RLC★★★

La subestación de Endesa en Málaga alimenta un circuito paralelo RLC industrial a 230 V / 50 Hz formado por: una resistencia R = 46 Ω, una bobina de inductancia L = 183 mH y un condensador C = 27,7 µF, todos en paralelo.

📝 Ver resolución completa

Apartado a — Corrientes de rama

a) Calcula las corrientes de rama IR, IL e IC.

Apartado b — Corriente total y factor de potencia

b) Calcula la corriente total I aplicando la composición fasorial.

c) Determina el factor de potencia del circuito e indica su carácter.

\[IR\ = \ \frac{230}{46} = \ 5\ A\ \ \ \ IL\ = \ \frac{230}{57,5} = \ 4\ A\ \ \ \ IC\ = \ \frac{230}{115} = \ 2\ A\]

\[I\ = \ \sqrt{IR^2 + (IL - IC)^2} = \ \sqrt{25 + 4} = \ \sqrt{29} \approx \ 5,39\ A\]

\[cos\ \varphi\ = \ \frac{IR}{I} = \ \frac{5}{5,39} = \ 0,928\ \ \rightarrow \ \ INDUCTIVO\ (IL\ > \ IC)\]

→ I ≈ 5,39 A · cos φ = 0,928 · Carácter INDUCTIVO

⚠ En paralelo: tensión igual en todas las ramas. I = \(\sqrt{I_R^2 + (I_L - I_C)^2}\).

⚠ Carácter: IL > IC → inductivo. IC > IL → capacitivo.

Apartado d — Frecuencia de resonancia

Calcula la frecuencia f₀ a la que el factor de potencia del circuito paralelo sería la unidad (resonancia en paralelo).

En resonancia paralelo \(I_L = I_C\) → \(X_L = X_C\) → \(\omega L = 1/(\omega C)\) → \(\omega_0^2 =\) 1/(LC).

\[L\ = \ 183\ mH\ = \ 0,183\ H\ \ \cdot \ \ C\ = \ 27,7\ \mu F\ = \ 27,7 \times 10^{-6}\ F\]

\[f_0\ = \ 1/(2\pi\sqrt{LC})\ = \ 1/(2\pi\sqrt{0,183 \times 27,7 \times 10^{-6}})\ = \ 1/(2\pi \times 0,002252)\ \approx \ 70,7\ Hz\]

→ f₀ ≈ 70,7 Hz · A esa frecuencia IL = IC y cos φ = 1 (resonancia)

RLC serie con diagrama fasorial [Ponencia P4A]★★

En la instalación eléctrica de un edificio de oficinas de Granada se mide el circuito RLC serie con los siguientes valores: tensión de alimentación U = 100 V a 50 Hz, R = 200 Ω, L = 50 mH y C = 15 µF.

a) Calcula la impedancia Z, la intensidad I y el ángulo de desfase φ. Indica el carácter del circuito.

b) Dibuja el diagrama fasorial de tensión indicando UR, UL, UC y U.

📝 Ver resolución completa

Apartado a — Intensidad

Reactancias:

\[XL\ = \ 2\pi \cdot f \cdot L\ = \ 2\pi \times 50 \times 0,050\ = \ 15,71\ \Omega\]

\[XC\ = \ \frac{1}{2\pi \cdot f \cdot C} = \ \frac{1}{2\pi \times 50 \times 15 \times 10^{-6}} = \ 212,21\ \Omega\]

Reactancia neta e impedancia:

\[X\ = \ XL - XC\ = \ 15,71 - 212,21\ = \ - 196,5\ \Omega\ \ \rightarrow \ \ CAPACITIVO\]

\[Z\ = \ \sqrt{R^2 + X^2} = \ \sqrt{200^2 + 196,5^2} = \ \sqrt{78\ 612} \approx \ 280,4\ \Omega\]

Intensidad eficaz:

\[I\ = \ \frac{U}{Z} = \ \frac{100\ V}{280,4\ \Omega} \approx \ 0,357\ A\]

→ I ≈ 0,357 A · Carácter CAPACITIVO (XC >> XL)

Apartado b — Diagrama fasorial de tensión

Calcular las tensiones parciales:

\[UR\ = \ I \cdot R\ = \ 0,357 \times 200\ = \ 71,3\ V\ \ (en\ fase\ con\ I)\]

\[UL\ = \ I \cdot XL\ = \ 0,357 \times 15,71\ = \ 5,6\ V\ \ (adelanta\ 90{^\circ}\ a\ I)\]

\[UC\ = \ I \cdot XC\ = \ 0,357 \times 212,21\ = \ 75,7\ V\ \ (retrasa\ 90{^\circ}\ a\ I)\]

Descripción del diagrama fasorial:

Eje horizontal (referencia): UR = 71,3 V en fase con I.

Eje vertical arriba (+90°): UL = 5,6 V.

Eje vertical abajo (−90°): UC = 75,7 V.

UC − UL = 75,7 − 5,6 = 70,1 V (neto capacitivo hacia abajo).

Tensión total U: desde origen, resultante = \(\sqrt{U_R^2 + (U_C - U_L)^2} = \sqrt{71{,}3^2 + 70{,}1^2} \approx 99{,}9\) V \(\approx 100\) V ✓

→ \(U = \sqrt{U_R^2 + (U_C - U_L)^2} \approx 100\) V ✓ \(U_R\) horizontal, \(U_L\)↑, \(U_C\)↓, \(U\) resultante

⚠ En el diagrama fasorial: UR horizontal (en fase con I), UL vertical arriba, UC vertical abajo.

⚠ U NO es la suma aritmética: \(U = \sqrt{U_R^2 + (U_L - U_C)^2}\). Las tensiones están desfasadas.

⚠ Carácter capacitivo: UC > UL → el fasor resultante de reactancias apunta hacia abajo.

⚠ Ponencia P4A: pide explícitamente DIBUJAR el diagrama. Practicar la representación gráfica.

Corrección del factor de potencia [Ponencia D1 Pb.4]★

Resonancia en serie RLC★★★

En la instalación eléctrica de una central fotovoltaica de Andújar (Jaén) se dispone de un circuito RLC serie con R = 20 Ω, L = 50 mH y C = 20 µF conectado a una fuente de tensión alterna de valor eficaz U = 200 V y frecuencia variable.

a) Calcular la frecuencia de resonancia f₀ del circuito.

b) Demostrar que en resonancia la impedancia es mínima e igual a R, y que el factor de potencia es 1.

c) Calcular la intensidad y las tensiones en R, L y C en resonancia. Verificar que Uₗ = U꜀ y que la tensión en la bobina y en el condensador pueden superar la tensión de alimentación.

📝 Ver resolución completa

Apartado a — Frecuencia de resonancia

\[En\ resonancia:\ X_l\ = \ X\ \ \Rightarrow \ \ \omega _0L\ = \ 1/(\omega _0C)\ \ \Rightarrow \ \ \omega _0\ = \ 1/\sqrt{LC}\]

\[\omega _0\ = \ 1/\sqrt{50 \cdot 10^{-3}\ \times \ 20 \cdot 10^{-6}}\ = \ 1/\sqrt{10^{-6}}\ = \ 1/10^{-3}\ = \ 1000\ rad/s\]

\[f_0\ = \ \omega _0\ /\ (2\pi)\ = \ 1000\ /\ (2\pi)\ = \ 159,15\ Hz\ \approx \ 159,2\ Hz\]

Apartado b — Impedancia y factor de potencia en resonancia

\[X_l\ = \ \omega _0 \cdot L\ = \ 1000 \times 50 \cdot 10^{-3}\ = \ 50\ \Omega\ \ \ \ \ X\ = \ 1/(\omega _0 \cdot C)\ = \ 1/(1000 \times 20 \cdot 10^{-6})\ = \ 50\ \Omega\]

\[Z\ = \ \sqrt{R^2\ + \ (X_l\ - \ X)^2}\ = \ \sqrt{20^2\ + \ (50 - 50)^2}\ = \ \sqrt{400\ + \ 0}\ = \ 20\ \Omega\ = \ R\ \ ✓\]

\[cos\varphi\ = \ R/Z\ = \ 20/20\ = \ 1\ \ \Rightarrow \ \ \varphi\ = \ 0{^\circ}\ \ (circuito\ puramente\ resistivo\ en\ resonancia)\ \ ✓\]

Apartado c — Intensidad y tensiones parciales en resonancia

\[I_0\ = \ U/Z\ = \ 200/20\ = \ 10\ A\ \ (intensidad\ máxima:\ Z\ mínima)\]

\[Uᴿ\ = \ I_0 \cdot R\ = \ 10 \times 20\ = \ 200\ V\ = \ U\ \ (toda\ la\ tensión\ cae\ en\ R)\]

\[U_l\ = \ I_0 \cdot X_l\ = \ 10 \times 50\ = \ 500\ V\ \ \ \ \ U\ = \ I_0 \cdot X\ = \ 10 \times 50\ = \ 500\ V\]

\[U_l\ = \ U\ = \ 500\ V\ \gg \ U\ = \ 200\ V\ \ \Rightarrow \ \ efecto\ de\ sobretensión\ (factor\ Q\ = \ X_l/R\ = \ 50/20\ = \ 2,5)\]

→ f₀ = 159,2 Hz · Z₀ = R = 20 Ω · cosφ = 1 · I₀ = 10 A · Uₗ = U꜀ = 500 V

⚠ En resonancia: Xₗ = X꜀, Z = Zmín = R, cosφ = 1, I = Imáx. Las tensiones en L y C se anulan entre sí en el circuito serie pero individualmente pueden ser mucho mayores que U (sobretensión, factor de calidad Q = Xₗ/R).

Circuito doméstico lámparas + motor — Condensador FdP [Ponencia 2025-26, P6]★★★

Un circuito doméstico de 230 V eficaces a 50 Hz alimenta dos lámparas de 75 W con un factor de potencia unidad y un motor que consume 500 VA con un factor de potencia 0,78 inductivo.

a) Dibujar el circuito, representando cada carga mediante una impedancia e incluyendo el interruptor de cada carga.

b) Determinar la corriente total cuando todas las cargas operan de forma simultánea.

c) ¿Qué condensador conectado en paralelo con las cargas dará un factor de potencia unidad?

📝 Ver resolución completa

Apartado a — Circuito

Las tres cargas (2 lámparas + 1 motor) se conectan en paralelo a la red de 230 V, 50 Hz. Cada rama lleva su interruptor. Las lámparas se modelan como resistencias puras (fdp = 1). El motor se modela como una impedancia RL (fdp = 0,78 inductivo).

Apartado b — Corriente total

Lámparas (resistivas, fdp = 1):

\[P_{\text{lamp}} = 2 \times 75 = 150 \text{ W} \qquad Q_{\text{lamp}} = 0 \text{ var}\]

Motor (fdp = 0,78 inductivo):

\[P_{\text{motor}} = S \cdot \cos\varphi = 500 \times 0{,}78 = 390 \text{ W}\]

\[Q_{\text{motor}} = S \cdot \sin\varphi = 500 \times \sin(\arccos 0{,}78) = 500 \times 0{,}6258 = 312{,}9 \text{ var}\]

Totales:

\[P_T = 150 + 390 = 540 \text{ W} \qquad Q_T = 312{,}9 \text{ var}\]

\[S_T = \sqrt{P_T^2 + Q_T^2} = \sqrt{540^2 + 312{,}9^2} = 624{,}1 \text{ VA}\]

\[\boxed{I_T = \frac{S_T}{V} = \frac{624{,}1}{230} = 2{,}71 \text{ A}}\]

Apartado c — Condensador para fdp = 1

Para conseguir fdp = 1, el condensador debe compensar toda la potencia reactiva:

\[Q_C = Q_T = 312{,}9 \text{ var}\]

\[C = \frac{Q_C}{\omega \cdot V^2} = \frac{312{,}9}{2\pi \times 50 \times 230^2} = \frac{312{,}9}{16{,}619 \times 10^6}\]

\[\boxed{C = 18{,}8 \;\mu\text{F}}\]

→ IT = 2,71 A · C = 18,8 μF para fdp = 1

⚠ Tipo Ponencia P6-D1: cargas en paralelo con distinto fdp. La clave es sumar P y Q por separado (método de Boucherot) y después calcular ST e IT. Para compensar a fdp = 1, QC = QT (se anula toda la reactiva inductiva). Fórmula: \(C = Q_C/(\omega \cdot V^2)\).

Tres cargas en paralelo con magnetotérmico [Ponencia 2025-26, P9]★★

La instalación eléctrica de un pequeño local comercial se conecta a la red monofásica de 230 V eficaces y 50 Hz, y queda protegida por un interruptor magnetotérmico bipolar (KM) situado a la entrada del cuadro. Aguas abajo del magnetotérmico se conectan en paralelo tres cargas: una primera carga de 1 200 W con factor de potencia 0,8 inductivo, una segunda carga puramente resistiva de 2 400 W (factor de potencia unidad) y una tercera carga de 3 200 W con factor de potencia 0,9 inductivo.

a) Aplicando el método de Boucherot, determine la corriente mínima que debe soportar el magnetotérmico para no dispararse en condiciones normales de funcionamiento.

b) Calcule el factor de potencia del conjunto de la instalación.

📝 Ver resolución completa

Resolución

Método de Boucherot: sumamos P y Q de cada carga por separado.

Carga 1 (1 200 W, fdp = 0,8):

\[Q_1 = P_1 \cdot \tan\varphi_1 = 1\,200 \times \tan(36{,}87^{\circ}) = 1\,200 \times 0{,}75 = 900 \text{ var}\]

Carga 2 (2 400 W, fdp = 1): Q2 = 0 var.

Carga 3 (3 200 W, fdp = 0,9):

\[Q_3 = P_3 \cdot \tan\varphi_3 = 3\,200 \times \tan(25{,}84^{\circ}) = 3\,200 \times 0{,}4843 = 1\,549{,}8 \text{ var}\]

Totales:

\[P_T = 1\,200 + 2\,400 + 3\,200 = 6\,800 \text{ W}\]

\[Q_T = 900 + 0 + 1\,549{,}8 = 2\,449{,}8 \text{ var}\]

\[S_T = \sqrt{6\,800^2 + 2\,449{,}8^2} = \sqrt{46{,}24 \times 10^6 + 6{,}00 \times 10^6} = 7\,227{,}8 \text{ VA}\]

⚠ Tipo Ponencia P9-D1: cargas en paralelo protegidas por magnetotérmico. Método de Boucherot: \(P_T = \sum P\), \(Q_T = \sum Q\), \(S_T = \sqrt{P_T^2 + Q_T^2}\). El magnetotérmico debe soportar \(I \geq S_T/V\). El fdp total NO es la media de los fdp individuales.
