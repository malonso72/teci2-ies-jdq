1
Introducción a los sistemas automáticos
📹 Teoría de control: conceptos generales

Un sistema automático de control es un conjunto de componentes que trabajan coordinadamente para que una magnitud física (temperatura, velocidad, nivel, presión…) alcance y mantenga un valor deseado, sin intervención humana continua.

Variables del sistema
Entrada (referencia/consigna): valor deseado que fija el operador.
Salida: valor real de la magnitud controlada.
Perturbación: factor externo no deseado que altera el sistema.
ℹ️
Ejemplos: termostato (T deseada → T real), riego automático (humedad deseada → humedad real), control de velocidad de crucero, puerta automática, semáforo programado.
2
Sistemas en lazo abierto y cerrado

2.1 Lazo abierto

La salida no influye en el funcionamiento del sistema: no hay rama de realimentación y la actuación se calcula solo a partir de la consigna.

Sistema de control en lazo abierto (diagrama normalizado).
Señales y bloques del diagrama
  • r — consigna. Valor deseado introducido en el sistema; única información que recibe el controlador.
  • G1 — controlador. Recibe r y genera la variable manipulada m. Sus parámetros se fijan por calibración.
  • m — variable manipulada. Acción que el controlador aplica sobre la planta (tensión, apertura, tiempo de encendido…).
  • G2 — planta o proceso. Sistema físico sobre el que actúa m (engloba al actuador).
  • c — variable controlada (salida). Respuesta del sistema. No se mide ni vuelve al controlador: no hay bloque H ni comparador.
Consecuencias. El sistema actúa solo en función de r y no conoce el valor real de c. No puede adaptarse a perturbaciones p ni corregir desviaciones. Su precisión depende exclusivamente de la calibración.

2.2 Lazo cerrado (realimentado)

La salida influye en el funcionamiento del sistema: el valor real de c se mide y se compara con la consigna r, y la diferencia (el error e) se usa para corregir la actuación.

Sistema de control en lazo cerrado (diagrama normalizado de la ponencia).
Señales del diagrama normalizado
  • r — consigna o referencia. Valor deseado (p. ej. 200 °C en un horno).
  • e — error o señal actuante. \(e = r - b\). Si e = 0, el sistema ha alcanzado el objetivo.
  • m — variable manipulada (señal de control). Salida del regulador; actúa sobre la planta.
  • c — variable controlada (salida). Magnitud física que se controla: temperatura, velocidad, nivel…
  • b — señal realimentada. Medida de c convertida por H a la misma naturaleza que r. Cumple \(b = H\cdot c\).
  • p — perturbación. Señal externa no controlada que actúa sobre la planta y modifica c (pendientes, viento, Tª ambiente, rozamiento…).
Bloques del diagrama normalizado
  • G1 — regulador o controlador. Recibe el error e y decide cómo corregirlo, generando m. Tipos P, I, D, PI, PD, PID (cap. 7).
  • G2 — planta o proceso. Sistema físico (horno, depósito, motor…). Engloba al actuador. Sobre él inciden m y p.
  • H — elementos de medida (sensor). Bloque de realimentación. Mide c y la convierte en b.
Cómo funciona el lazo. El sensor H mide c y produce b. El comparador calcula \(e = r - b\). Si \(e\neq 0\), el regulador G1 genera m que actúa sobre G2; la salida c cambia y con ella b, reduciendo el error. El bucle actúa en el sentido de llevar e a cero, y con él c al valor deseado r.
Por qué hace falta el bloque H. r y c suelen ser de distinta naturaleza: r puede ser una tensión eléctrica (potenciómetro, teclado) y c una temperatura, velocidad o posición. Para poder compararlas, H transforma c en una señal b del mismo tipo que r. Matemáticamente, \(b = H\cdot c\). En el caso particular de realimentación unitaria (H = 1) se tiene b = c y \(e = r - c\).

2.4 Error y realimentación

Error = Consigna − Señal realimentada → \(e = r - b\), con \(b = H\cdot c\).
La realimentación negativa reduce el error (estabiliza). La positiva lo aumenta (desestabiliza).
Desarrollo del concepto de error

El error es la diferencia entre la consigna r y la señal realimentada b (medida de la salida c a través de H):

\(e = r - b,\quad b = H\cdot c\)

En el caso particular de realimentación unitaria (\(H = 1\)) se tiene \(b = c\) y la fórmula se simplifica:

\(e = r - c\)
  • \(r\) es la consigna o referencia.
  • \(c\) es la variable controlada (salida real).
  • \(b\) es la señal realimentada, \(b = H\cdot c\).
  • \(e\) es el error.

Interpretación:

  • \(e = 0\) → la salida coincide con la referencia.
  • \(e > 0\) → la salida es menor que la referencia.
  • \(e < 0\) → la salida es mayor que la referencia.

El objetivo de un sistema de control es minimizar el error. Los reguladores actúan precisamente sobre él: la acción proporcional reduce el error instantáneo, la integral ayuda a eliminar el error permanente y la derivativa anticipa cambios y reduce oscilaciones.

2.5 Dos formas de dibujar el lazo de control

En la bibliografía existen dos convenciones habituales para dibujar el mismo lazo de control:

  • Representación física: se identifica cada elemento real del sistema por separado (transductor de entrada, regulador, actuador y captador). Es la forma más didáctica y la que encontrarás en libros de automatización industrial y en problemas de PEvAU que piden identificar componentes.
  • Representación normalizada: se agrupan los elementos en los bloques \(G_1\), \(G_2\) y \(H\) para aplicar las reglas de álgebra de bloques. Es la forma que necesitas para calcular la función de transferencia en lazo cerrado.

Ambas son equivalentes y describen el mismo sistema. Usa la forma A para identificar elementos reales y la forma B para hacer cálculos.

★
Equivalencia entre las dos formas:
G₁ (rama directa) = Transductor · Regulador · Actuador
G₂ (planta) = Planta o proceso
H (realimentación) = Captador (sensor + acondicionamiento)
m (variable manipulada) = acción del Regulador+Actuador sobre la Planta
3
Elementos de un sistema de control
1
Consigna / Referencia (r): valor deseado que introduce el operador.
2
Comparador: calcula el error \(e = r - \text{señal realimentada}\). Símbolo: círculo con ×, signos ± en sus entradas.
3
Controlador / Regulador: procesa el error y genera la señal de control (P, PI, PD, PID).
4
Actuador: ejecuta la acción física sobre la planta (motor, válvula, resistencia…).
5
Planta / Sistema: proceso físico que se desea controlar.
6
Sensor / Transductor: mide la variable de salida y la convierte en señal comparable con la referencia.
7
Perturbación (p): señal externa no controlada que actúa sobre la planta y modifica la variable controlada c. No es un bloque del sistema sino una entrada adicional. Ejemplos: cambios de Tª ambiente en un horno, pendientes o viento en un crucero, variaciones de carga mecánica en un motor, rozamientos. El objetivo del control en lazo cerrado es precisamente compensar el efecto de las perturbaciones para que c siga a r.

Representación: diagramas de bloques

Representación gráfica del flujo de señales en un sistema de control. Elementos básicos:

●
Bloques funcionales: rectángulos con la función de transferencia G dentro.
●
Flechas: indican la dirección del flujo de la señal.
●
Comparadores / puntos de suma: círculo con × y signos ± en cada entrada.
●
Puntos de bifurcación: donde la señal se divide hacia dos o más destinos.
★
Realimentación negativa: la señal se resta → reduce error → estabiliza.
Realimentación positiva: la señal se suma → aumenta error → desestabiliza.
ⓘ
Sobre las perturbaciones (p): en un diagrama de bloques las perturbaciones no son un bloque, sino una señal externa que se suma a la entrada de la planta G2 (o de algún elemento intermedio). Se representan con una flecha entrante adicional y suelen marcarse en color o con etiqueta p. El lazo cerrado actúa para que su efecto sobre c sea mínimo.
5
Función de transferencia y álgebra de bloques
Función de transferencia [F1]
\(G = \dfrac{\text{Salida}}{\text{Entrada}}\)

Indica cómo responde el sistema ante una señal de entrada en régimen permanente.

Regla 1 — Asociación en serie [F2]

Regla 1: bloques en serie — la salida de uno es la entrada del siguiente.
\(G_{eq} = G_1 \cdot G_2 \cdot \ldots \cdot G_n\)

Regla 2 — Asociación en paralelo [F3]

Regla 2: bloques en paralelo — misma entrada, salidas sumadas.
\(G_{eq} = G_1 + G_2 + \ldots + G_n\)

Si alguna rama llega con signo −, su función se resta.

Regla 3 — Realimentación negativa ⭐ [F4]

Regla 3: lazo cerrado con realimentación negativa — fórmula fundamental.
Fórmula fundamental del lazo cerrado
\(G_{\text{eq}} = \dfrac{G}{1 + G \cdot H}\)

Demostración: \(S = G \cdot e\), \(e = R - H \cdot S\) → \(S = G(R - H \cdot S)\) → \(S(1 + GH) = GR\) → \(G_{eq} = S/R = \dfrac{G}{1 + GH}\)

Regla 4 — Realimentación positiva [F5]

Regla 4: lazo cerrado con realimentación positiva.
\(G_{\text{eq}} = \dfrac{G}{1 - G \cdot H}\)

⚠ Tiende a aumentar el error. Si G·H = 1 → denominador = 0 → sistema inestable.

Regla 5 — Realimentación unitaria negativa (H = 1) [F6]

Regla 5: la salida vuelve directamente al comparador sin pasar por ningún bloque.
\(G_{\text{eq}} = \dfrac{G}{1 + G}\)

Caso particular de R3 con H = 1. Es el caso más habitual en la práctica.

Regla 6 — Bloque exterior al lazo (precascada) [F7]

Regla 6: G₁ exterior al lazo formado por G₂ y H. G₁ no entra en el denominador.
\(G_{\text{eq}} = \dfrac{G_1 \cdot G_2}{1 + G_2 \cdot H}\)

\(G_1\) solo escala la señal de referencia antes del comparador, nunca aparece dentro del lazo.

Regla 7 — Fusión de sumadores consecutivos [F8] Ampliación · no entra en PEvAU 2025-26

Regla 7: dos o más sumadores consecutivos se fusionan en uno solo.
\(\text{Salida} = R + D_1 - D_2\) (conservando signos)

Aparece en diagramas con perturbaciones que entran en distintos puntos del sistema.

Regla 8 — Bloque inverso (H = 1/G) [F9] Ampliación · no entra en PEvAU 2025-26

Regla 8: cuando H = 1/G, la ganancia se reduce siempre a la mitad.
\(G_{\text{eq}} = \dfrac{G}{2}\)

Si \(H = 1/G\) → \(G \cdot H = 1\) → denominador \(= 2\) siempre. Ejemplo: \(G = 10\), \(H = 0{,}1\) → \(G_{eq} = 5\).

Regla 9 — Lazos anidados Ampliación · no entra en PEvAU 2025-26

Regla 9: simplificar siempre desde el lazo más interno hacia el más externo.
Paso 1: \(G_2\prime = \dfrac{G_2}{1 + G_2 \cdot H_2}\) → Paso 2: \(G_{\text{serie}} = G_1 \cdot G_2\prime\) → Paso 3: \(G_{eq} = \dfrac{G_{\text{serie}}}{1 + G_{\text{serie}} \cdot H_1}\)

Cada lazo aporta un término al denominador de la función de transferencia final.

5.11 Procedimiento general de simplificación

Para simplificar cualquier diagrama de bloques se aplica el siguiente procedimiento ordenado:

  1. Analizar el diagrama: identificar la entrada, la salida, los bloques, los comparadores y los lazos de realimentación.
  2. Simplificar bloques en serie: multiplicar sus funciones de transferencia.
  3. Simplificar bloques en paralelo: sumar sus funciones de transferencia respetando signos.
  4. Simplificar lazos de realimentación aplicando \(G_{eq} = G / (1 \pm G\cdot H)\), empezando siempre por el lazo más interno.
  5. Repetir hasta obtener un único bloque equivalente.
  6. Sustituir los valores numéricos y calcular el resultado.
⭐
Importante para PEvAU. En el examen se piden normalmente dos pasos distintos: (1) obtener la expresión algebraica de \(G_{eq}\) en función de \(G_1\), \(G_2\), \(H\)…, y (2) sustituir los valores numéricos para calcular el resultado. Nunca sustituir antes de simplificar algebraicamente.

5.12 Cálculo del error en lazo cerrado

En un sistema con realimentación negativa, el error se define como la diferencia entre la referencia y la señal realimentada:

Definición del error
\(e = r - H \cdot c\)   con   \(c = G_{eq} \cdot r\)

Si se conoce la función de transferencia equivalente \(G_{eq}\), la salida vale \(c = G_{eq}\cdot r\). Combinando ambas expresiones se calcula el error para cualquier entrada \(r\).

Ejemplo resuelto. Sistema con \(G = 6\), \(H = 0{,}5\) (realimentación negativa) y entrada \(r = 10\):
  • \(G_{eq} = G/(1+G\cdot H) = 6/4 = 1{,}5\)
  • \(c = G_{eq} \cdot r = 1{,}5 \cdot 10 = 15\)
  • \(e = r - H\cdot c = 10 - 0{,}5\cdot 15 = \boxed{2{,}5}\)

Estabilidad y error →
Comparación: sistema estable (oscilaciones amortiguadas) vs inestable (oscilaciones crecientes).

✔ Sistema estable

La salida oscila inicialmente pero se amortigua y tiende a un valor constante (equilibrio). Ejemplo: termostato que alcanza y mantiene la temperatura deseada.

✘ Sistema inestable

Las oscilaciones crecen con el tiempo. El sistema no alcanza equilibrio. Ejemplo: control mal ajustado con vibraciones cada vez mayores.

⚠️
Condición de inestabilidad: si el denominador (1 + G·H) se aproxima a 0, la ganancia tiende a infinito → sistema inestable. La realimentación negativa estabiliza; la positiva desestabiliza.
Comportamiento frente a perturbaciones: un sistema estable absorbe la perturbación y vuelve al equilibrio. Un sistema inestable amplifica la perturbación hasta destruirse o saturarse.

Ejemplo tipo PEvAU — Identificación de la estabilidad

Un sistema de control de temperatura tiene realimentación negativa con \(G = 5\) y \(H = 0{,}5\). ¿Es estable el sistema? Justifica la respuesta.

Resolución.
  • Producto \(G\cdot H = 5\cdot 0{,}5 = 2{,}5\)
  • Aplicando realimentación negativa: \(G_{eq} = \dfrac{G}{1 + G\cdot H} = \dfrac{5}{1 + 2{,}5} = \dfrac{5}{3{,}5} \approx 1{,}43\)
  • El denominador \((1 + G\cdot H) = 3{,}5 > 0\) → el sistema es estable.
  • La realimentación negativa reduce la ganancia de 5 a 1,43 y estabiliza el sistema.
💡
Nota PEvAU. Las preguntas más frecuentes sobre estabilidad en Andalucía son: (1) indicar si un sistema es estable o inestable y justificarlo cualitativamente; (2) explicar por qué la realimentación negativa estabiliza y la positiva puede desestabilizar; (3) relacionar la estabilidad con el valor del producto \(G\cdot H\) en el denominador de la función de transferencia.

Reguladores PID →
📹 Controlador PID 📹 Acciones básicas (P, I, D, PD, PI, PID)

El regulador o controlador es el "cerebro" del lazo de control. Procesa la señal de error y genera la acción de control m(t) (variable manipulada) que envía al actuador. Según cómo procesa el error, distinguimos seis tipos de acción.

Acción Proporcional (P)

\(m(t) = K_p \cdot e(t)\) — Respuesta proporcional al error actual.
Respuesta del regulador P.
Diagrama de lazo cerrado — Regulador P.

Ventaja: rápido y sencillo. Limitación: deja error en régimen permanente (offset).

EJEMPLO REAL — CALEFACCIÓN CON REGULADOR P

Quieres mantener la habitación a 22 °C. Consigna: 22 °C. Temperatura actual: 18 °C.

\(e = 22 - 18 = 4\) °C. Con \(K_p = 25\%\) de potencia por cada grado → \(m = 25 \times 4 = 100\%\). La calefacción se pone a tope.

La habitación se calienta: \(T = 21\) °C, \(e = 1\) °C → \(m = 25\%\). La calefacción baja. Cuando llega a 22 °C: \(e = 0\), \(m = 0\).

¿Por qué queda un error permanente? Porque el proporcional solo actúa si hay error. Si llegara exactamente a 22 °C, el error sería cero y la calefacción se apagaría; la temperatura volvería a bajar. Siempre queda un pequeño error residual (offset).

Acción Integral (I)

\(m(t) = K_i \displaystyle\int e(t)\,dt\) — Acumula el error a lo largo del tiempo.
Respuesta del regulador I.
Diagrama de lazo cerrado — Regulador I.

Ventaja: elimina el error permanente. Limitación: respuesta lenta, puede provocar sobreoscilaciones.

EJEMPLO REAL — CALEFACCIÓN CON REGULADOR I

Consigna: 22 °C. \(T_{\text{actual}} = 20\) °C, \(e = 2\) °C. El regulador integral empieza a acumular el error y actuar.

Pasa el tiempo: \(T = 21{,}5\) °C, \(e = 0{,}5\) °C. El integral sigue actuando porque el error aún existe.

\(T = 21{,}8\) °C, \(e = 0{,}2\) °C. El regulador todavía empuja. Finalmente: \(T = 22\) °C, \(e = 0\). El integral deja de crecer, pero el valor acumulado mantiene la calefacción justo donde hace falta.

¿Por qué puede pasarse y oscilar? Porque el integral sigue acumulando mientras hay error, y cuando por fin llega al valor deseado, ya lleva un exceso de acción acumulada que puede hacer que se pase al otro lado.

Acción Derivativa (D)

\(m(t) = K_d \cdot \dfrac{de(t)}{dt}\) — Reacciona a la velocidad de cambio del error.
Respuesta del regulador D.
Diagrama de lazo cerrado — Regulador D.

Ventaja: anticipa cambios y amortigua oscilaciones. Limitación: no actúa si el error es constante, amplifica ruido.

EJEMPLO REAL — CALEFACCIÓN CON REGULADOR D

Consigna: 22 °C. \(T_{\text{actual}} = 18\) °C, error grande. El sistema calienta fuerte.

La temperatura sube muy rápido: 20 °C → 21 °C. El regulador derivativo detecta: te acercas demasiado rápido. Reduce la potencia antes de llegar → frena el calentamiento → no se pasa de 22 °C.

¿Por qué nunca se usa solo? Porque si el error es constante (aunque sea grande), el derivativo vale cero y no hace nada. Necesita combinarse con P (o con P + I) para ser útil.

Regulador PI (Proporcional + Integral)

\(m(t) = K_p \cdot e(t) + K_i \displaystyle\int e(t)\,dt\)
Respuesta del regulador PI.
Diagrama de lazo cerrado — Regulador PI.

Combina rapidez (P) + eliminación del offset (I). Es el regulador más usado cuando no hay cambios bruscos.

EJEMPLO REAL — CALEFACCIÓN CON REGULADOR PI

Consigna: 22 °C. \(T_{\text{actual}} = 18\) °C, \(e = 4\) °C.

Parte P actúa primero con fuerza: error grande → calefacción fuerte → temperatura sube rápido.

\(T = 21\) °C, \(e = 1\) °C: P reduce la potencia. Pero aún hay error. La parte I lleva tiempo acumulando ese error y sigue empujando. Añade más potencia lentamente. No para hasta que el error sea cero.

\(T\) sigue subiendo: \(21{,}5 \to 21{,}8 \to 21{,}9\) °C. Hasta que: \(22\) °C, \(e = 0\). El integral se congela con su valor acumulado y mantiene la calefacción estable.

P arranca rápido · I termina el trabajo · juntos: rápido y sin error permanente.

Regulador PD (Proporcional + Derivativo)

\(m(t) = K_p \cdot e(t) + K_d \cdot \dfrac{de(t)}{dt}\)
Respuesta del regulador PD.
Diagrama de lazo cerrado — Regulador PD.

Combina rapidez (P) + anticipación (D). No elimina el error permanente.

EJEMPLO REAL — CALEFACCIÓN CON REGULADOR PD

Consigna: 22 °C. T actual = 18 °C, e = 4 °C.

Parte P actúa con fuerza: calefacción fuerte, temperatura sube rápido. A 20 °C sube rápido, a 21 °C sube muy rápido.

Parte D detecta: te acercas demasiado rápido. Reduce la potencia antes de llegar → frena el calentamiento.

La temperatura llega suave: 21,8 → 21,9 °C. Se estabiliza cerca de 22 °C, pero no llega exacto. Para eso haría falta el integrador.

P arranca rápido · D frena antes de pasarse · juntos: rápido y sin oscilaciones · no elimina el error permanente.

Regulador PID (Proporcional + Integral + Derivativo)

\(m(t) = K_p \cdot e(t) + K_i \displaystyle\int e(t)\,dt + K_d \cdot \dfrac{de(t)}{dt}\)
Respuesta del regulador PID.
Diagrama de lazo cerrado — Regulador PID.
★
PID = respuesta rápida (P) + sin error permanente (I) + anticipación y amortiguamiento (D). Es el regulador más completo y más usado en la industria. Su ajuste es el más complejo.

Ejemplo real — Calefacción con regulador PID

EL TERMOSTATO PERFECTO

Consigna: 22 °C. T actual = 18 °C, e = 4 °C.

Al principio domina el P: calefacción fuerte, sube rápido.

A medida que se acerca, el D cobra protagonismo: detecta que se acerca muy rápido → reduce potencia → evita que se pase.

T = 21,8 → 21,9 °C (llega suave). Con el error ya pequeño, el I hace el trabajo fino: detecta el pequeño error residual → añade potencia poco a poco → no para hasta llegar a cero.

Resultado: T = 22 °C exactos. Sistema estable.

P arranca · D frena · I ajusta hasta cero · el mejor de los tres mundos.

NoNoP — cuando la precisión absoluta no es crítica. Ej: velocidad de un ventilador, brillo de una lámpara.
SíNoPI — el más común en la industria. Ej: presión en tuberías, velocidad de motores industriales, caudal de líquidos.
NoSíPD — cuando importa no pasarse. Ej: estabilizador de cámara, posicionamiento rápido de cabezal.
SíSíPID — el estándar industrial. Ej: horno industrial, dron en vuelo, control de inyección de combustible, robot industrial.
ℹ️
Nota: el regulador D solo nunca tiene sentido en la práctica: no corrige error permanente y no reacciona a errores constantes. Siempre se combina con P o con P + I.

Ejemplos resueltos

Ejemplo 1 — Cálculo de la señal de control (regulador P).
Un sistema dispone de un regulador proporcional con ganancia \(K_p = 5\). En un determinado instante el error es \(e(t) = 2\). Calcula la señal de control \(m(t)\).
Resolución: Aplicando la expresión del regulador proporcional:
\(m(t) = K_p \cdot e(t) = 5 \cdot 2 = 10\)
La señal de control en ese instante es \(m(t) = 10\) unidades.
Ejemplo 2 — Justificación de la elección de regulador (pregunta conceptual).
En una instalación industrial se controla la temperatura de un horno. Se observa que con un regulador proporcional (P) se alcanza rápidamente la temperatura deseada, pero existe un error permanente. ¿Qué regulador debería utilizarse para eliminarlo? Justifica tu respuesta.
Resolución:
Se debe sustituir el regulador P por un regulador PI (proporcional-integral).
La acción proporcional mantiene la respuesta rápida del sistema.
La acción integral acumula el error a lo largo del tiempo y sigue actuando hasta que el error sea nulo, lo que elimina el error permanente.
Un regulador solo integral (I) también eliminaría el error, pero la respuesta sería demasiado lenta para una aplicación industrial.
Por tanto, el regulador PI es la solución más adecuada en este caso.

Actuadores
📌
Posición en el sistema. El actuador se sitúa entre el regulador y la planta. Recibe la señal de control \(m(t)\) generada por el regulador y la transforma en una acción física (movimiento, calor, caudal…) que modifica la variable de salida \(c(t)\).

9.1 Concepto de actuador

Un actuador es el elemento del sistema de control encargado de transformar la señal de control procedente del regulador en una acción física sobre el sistema. Actúa directamente sobre la planta modificando el valor de la variable controlada.

Ejemplos rápidos:

  • En un sistema de calefacción, el actuador puede ser una resistencia eléctrica que produce calor.
  • En el control de velocidad de un motor, el actuador es el propio motor eléctrico.
  • En un sistema hidráulico, el actuador puede ser una válvula que regula el caudal.

9.2 Motores eléctricos

Transforman la energía eléctrica en movimiento mecánico de rotación. Son uno de los actuadores más utilizados en sistemas de control. La velocidad o el par del motor pueden regularse mediante la señal de control \(m(t)\) procedente del regulador.

Tipos principales:

  • Motor de corriente continua (CC): control sencillo y preciso de la velocidad. Sistemas de velocidad variable.
  • Motor de corriente alterna (CA): robusto y económico, de uso general en industria y electrodomésticos.
  • Servomotor: control preciso de posición y velocidad. Robótica y máquinas CNC.
  • Motor paso a paso: gira en pasos discretos. Posicionamiento exacto sin sensor. Impresoras 3D.

9.3 Cilindros neumáticos e hidráulicos

Transforman la energía de un fluido a presión en movimiento mecánico lineal.

  • Cilindros neumáticos: utilizan aire comprimido. Movimientos rápidos en automatización industrial.
  • Cilindros hidráulicos: utilizan aceite a presión, lo que permite generar fuerzas mucho mayores. Maquinaria pesada y prensas.

El movimiento del cilindro se controla mediante válvulas que regulan el paso del fluido.

9.4 Resistencias calefactoras

Transforman la energía eléctrica en energía térmica mediante el efecto Joule. La potencia térmica generada depende de la señal de control aplicada, lo que permite regular la temperatura del sistema.

Aplicaciones: calefacción, hornos eléctricos, termos, secadoras, climatización industrial.

9.5 Válvulas de control

Regulan el paso de fluidos en sistemas neumáticos, hidráulicos o térmicos. La apertura puede controlarse mediante señales eléctricas, neumáticas o hidráulicas.

Variando la apertura se controla el caudal, la presión, el nivel o la temperatura del sistema.

9.6 Otros actuadores

  • Electroválvula: válvula accionada eléctricamente para control todo/nada del paso de fluidos.
  • Relé / contactor: controla circuitos de potencia mediante señales eléctricas débiles.
  • Bomba: transforma energía eléctrica en movimiento de fluidos. Circuitos hidráulicos y de agua.
  • Ventilador: genera movimiento de aire para temperatura o ventilación.
  • Lámpara: transforma energía eléctrica en luz. Señalización e iluminación automática.

9.8 Ejemplo tipo PEvAU

Identificación de actuadores según la aplicación. Indique qué actuador utilizaría en cada caso y justifique brevemente la respuesta:

  1. Regular la temperatura de un horno industrial.
    → Resistencia calefactora. Transforma la señal eléctrica en calor mediante el efecto Joule, permitiendo regular la temperatura.
  2. Desplazar una pieza en una línea de montaje automática.
    → Cilindro neumático. Convierte el aire comprimido en movimiento lineal, ideal para desplazamientos rápidos en automatización.
  3. Controlar el caudal de agua en una tubería.
    → Válvula de control. Regula el paso del fluido variando su apertura en función de la señal de control.
  4. Posicionar con precisión el brazo de un robot.
    → Servomotor. Permite un control preciso de posición y velocidad, diseñado para aplicaciones robóticas.

Resumen final y estrategia de resolución
★
Alcance del bloque: serie, paralelo, realimentación negativa/positiva, unitaria, bloque exterior al lazo. No se exigen cambios de puntos de bifurcación/suma (R7–R9) ni criterio de Routh. Las R7–R9 se incluyen como formación complementaria.
Fórmulas imprescindibles
[F1] \(G = \dfrac{S}{E}\)  ·  [F2] Serie: \(G_{eq} = G_1 \cdot G_2\)  ·  [F3] Paralelo: \(G_{eq} = G_1 + G_2\)
[F4] Realim. (−): \(G_{eq} = \dfrac{G}{1 + GH}\)  ·  [F5] Realim. (+): \(G_{eq} = \dfrac{G}{1 - GH}\)
[F6] Unitaria (−): \(G_{eq} = \dfrac{G}{1 + G}\)  ·  [F7] Precascada: \(G_{eq} = \dfrac{G_1 G_2}{1 + G_2 H}\)
[F8] Fusión sumadores  ·  [F9] Bloque inverso: \(G_{eq} = \dfrac{G}{2}\) (si \(H = 1/G\))

Estrategia para resolver problemas

1
Leer el enunciado e identificar: referencia, bloques, lazos, sensores, perturbaciones.
2
Si pide diagrama de bloques: dibujar con comparador, regulador, actuador, planta, sensor.
3
Si pide función de transferencia: simplificar algebraicamente (serie → paralelo → lazo, de dentro a fuera).
4
Obtener \(G_{eq}\) simbólica primero, luego sustituir valores.
5
Si pide error: calcular \(c = G_{eq} \cdot r\) y luego \(e = r - H \cdot c\).
6
Si pide estabilidad: comprobar que el denominador no se anule (\(1 + GH \neq 0\)).
7
Si pide tipo de regulador: explicar P, I, D y justificar cuál conviene según el proceso.

📝 Decálogo PAU 2025-26

  1. Álgebra de bloques: serie \(G_{eq}=G_1\cdot G_2\); paralelo \(G_{eq}=G_1+G_2\); realimentación \(G_{eq}=\dfrac{G}{1\pm G\cdot H}\).
  2. Realimentación negativa: el signo del denominador es \(+\); positiva: signo \(-\).
  3. Ponencia Andalucía 2025-26: NO entra criterio de Routh-Hurwitz. Se pide estabilidad solo cualitativa.
  4. Lazo cerrado estabiliza si \(1 + G\cdot H \neq 0\): si se anula, el sistema es inestable (oscilación infinita).
  5. Lazo abierto es más simple y barato pero no corrige perturbaciones; cerrado es más preciso pero puede ser inestable.
  6. Regulador P: proporcional al error; reduce error pero no lo elimina. Regulador I: integral, elimina el error estacionario. Regulador D: derivativo, mejora respuesta transitoria.
  7. PID combina las tres acciones: \(m(t) = K_p\,e + K_i\!\int\! e\,dt + K_d\,\dot{e}\).
  8. Sensor: transforma magnitud física en señal eléctrica. Actuador: transforma señal eléctrica en acción física.
  9. Bifurcaciones y nudos suma NO entran como problema complejo, solo en diagramas básicos.
  10. Al simplificar un diagrama, respeta el orden: primero realimentaciones internas, luego serie/paralelo externos.
